# 付録A: よく使うフィールド早見表

Falcoルール作成時に頻出するフィールドを、カテゴリ別にまとめた早見表です。各フィールドの型、説明、利用場面、注意点を記載しています。

## プロセス関連フィールド

| フィールド名 | 型 | 説明 | よく使う場面 | 注意点 |
|---|---|---|---|---|
| `proc.name` | string | プロセス名（バイナリ名） | 特定コマンド検出、ホワイトリスト | basename のみ、フルパスではない |
| `proc.args` | string | プロセス引数（スペース区切り） | コマンドオプション検出 | 全引数を含む、サイズ制限あり |
| `proc.cmdline` | string | コマンドライン全体 | 詳細な実行検出 | proc.args より詳細、メモリ効率は低い |
| `proc.exe` | string | 実行ファイルのフルパス | パス検証、不正プロセス検出 | symlink解決済み |
| `proc.pname` | string | 親プロセス名 | 実行元判定（例：shellから実行） | 親プロセスがない場合は空 |
| `proc.ppid` | uint64 | 親プロセスID | 親プロセス判定 | PID reuse の可能性 |
| `proc.pid` | uint64 | プロセスID | ホワイトリスト（短期的） | 再利用される、永続判定は不可 |
| `proc.uid` | uint32 | 実効ユーザーID | 権限分離ルール | setuid の場合と異なる可能性 |

## ファイルディスクリプタ関連フィールド

| フィールド名 | 型 | 説明 | よく使う場面 | 注意点 |
|---|---|---|---|---|
| `fd.name` | string | ファイル/ソケット名 | ファイルパスフィルタ | パス長に制限あり |
| `fd.type` | string | ディスクリプタ種類 | ファイル/ソケット区別 | file, socket, ipv4, ipv6など |
| `fd.num` | int64 | ファイルディスクリプタ番号 | デバッグ用 | プロセス固有 |
| `fd.rip` | string | リモートIP（受信） | ネットワークフィルタ | ソケットのみ有効 |
| `fd.lip` | string | ローカルIP（受信） | ローカルインターフェース判定 | ソケットのみ有効 |

本書の詳細版では、さらに多くのフィールドについて、実践的な使用例とともに解説しています。
