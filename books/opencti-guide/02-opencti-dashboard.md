## 2.1 OpenCTIの概要

OpenCTIは、フランスのサイバーセキュリティ機関ANSSI（Agence nationale de la securite des systemes d'information）の発案で生まれ、現在はFiligran社が開発・メンテナンスを行うオープンソースの脅威インテリジェンスプラットフォームです。

一言で言えば、サイバー脅威に関するあらゆる情報を「入れて、つなげて、見渡す」ためのツールです。

## 2.2 何ができるのか

OpenCTIの主要な機能は以下の通りです。

**脅威情報の一元管理**: 複数のソース（MISP、AlienVault OTX、Abuse.ch、MITRE ATT&CKなど）から脅威情報を自動収集し、1つの場所に統合します。

**関連性の可視化**: 「このマルウェアはどの脅威アクターが使っているか」「この脆弱性はどの攻撃キャンペーンで悪用されたか」といった関連性をグラフで可視化できます。

**STIX 2.1ネイティブ**: 国際標準のデータフォーマットSTIX 2.1をネイティブにサポートしており、他のツールとの相互運用性が高いです。

**コネクタによる拡張**: 100以上の公式・コミュニティコネクタがあり、外部のデータソースやツールと簡単に接続できます。

**GraphQL API**: すべての機能がAPIから操作可能で、自動化やカスタム統合が容易です。

## 2.3 アーキテクチャ

OpenCTIは以下のコンポーネントで構成されています。

```
┌─────────────────────────────────────────────────┐
│                  OpenCTI Platform                │
│              (Node.js + React SPA)               │
├─────────────────────────────────────────────────┤
│                  GraphQL API                     │
├──────────┬──────────┬───────────┬───────────────┤
│  Redis   │ Elastic  │   MinIO   │   RabbitMQ    │
│ (Cache)  │ Search   │ (Storage) │  (Messaging)  │
└──────────┴──────────┴───────────┴───────────────┘
                        ↑
          ┌─────────────┼─────────────┐
          │             │             │
     ┌────┴────┐  ┌────┴────┐  ┌────┴────┐
     │ Worker  │  │ Worker  │  │ Worker  │
     │  (1)    │  │  (2)    │  │  (3)    │
     └────┬────┘  └────┬────┘  └────┬────┘
          │             │             │
     ┌────┴─────────────┴─────────────┴────┐
     │          Connectors                  │
     │  (MITRE, MISP, AlienVault, ...)      │
     └─────────────────────────────────────┘
```

各コンポーネントの役割は次の通りです。

| コンポーネント | 役割 |
|---------------|------|
| **Platform** | WebUI + GraphQL APIを提供するメインアプリケーション |
| **ElasticSearch** | 全文検索とデータのインデキシング |
| **Redis** | キャッシュとセッション管理 |
| **MinIO** | ファイル（レポートPDF、マルウェアサンプルなど）のオブジェクトストレージ |
| **RabbitMQ** | コネクタとWorker間の非同期メッセージング |
| **Worker** | バックグラウンドでデータの書き込み処理を実行 |
| **Connector** | 外部データソースとの接続・データ取り込み |

## 2.4 Community版とEnterprise版

OpenCTIにはCommunity（OSS）版とEnterprise版があります。

Community版は無料で利用でき、本ガイドで紹介する機能のほとんどが含まれています。Enterprise版では高度な分析機能、RBAC（ロールベースアクセス制御）、SLA付きサポートなどが追加されます。

デモ環境（demo.opencti.io）ではEnterprise版の機能も一部体験できます。

## 2.5 デモ環境へのアクセス

Filigranのアカウントを作成すると、デモ環境にアクセスできます。

- **URL**: https://demo.opencti.io
- **ログイン**: 「Login with Filigran」からFiligranアカウントでSSO認証
- **注意**: デモ環境は24時間ごとにリセットされます

> **デモ環境のスクリーンショット**: 本ガイドのスクリーンショットはすべてこのデモ環境から取得しています。同じ画面を確認したい場合は、デモ環境にログインして該当ページに移動してください。

## 2.6 ダッシュボード概要

ログイン後に最初に表示されるのがダッシュボードです。

> **確認方法**: https://demo.opencti.io/dashboard
> <!-- スクリーンショット: ダッシュボード画面 -->

ダッシュボードは、プラットフォーム全体の「今」を一目で把握するための画面です。

## 2.7 上部のサマリーカード

画面上部には4つのカードが並んでいます。

| カード | 意味 | デモ環境の例 |
|--------|------|-------------|
| **Intrusion Set** | 登録されている攻撃グループ数 | 1.31K |
| **Malware** | 登録されているマルウェア数 | 5.31K |
| **Report** | 脅威レポート数 | 6.34K |
| **Indicator** | IOC（侵害指標）の数 | 668.71K |

各カードの右上には「過去24時間の増減」が表示され、プラットフォームにどれだけ新しい情報が流入しているかが分かります。

## 2.8 Most Active Threats（直近3ヶ月）

横棒グラフで、最近最も活発な脅威が表示されます。デモ環境では以下のような脅威が上位に表示されています。

- **Hydra Saiga** — 最も活発
- **Clop** — ランサムウェアグループ
- **MuddyWater** — イラン系APTグループ
- **Lazarus Group** — 北朝鮮系APTグループ

このグラフを見ることで、「今、世界でどの攻撃グループが活発に活動しているか」を直感的に把握できます。

## 2.9 Most Targeted Victims（直近3ヶ月）

攻撃の標的となっている地域・業種を横棒グラフで表示します。

デモ環境では「Asia」「Europe」「Americas」が地域として上位に、「Government and administration」「Finance」「Technologies」「Energy」が業種として上位に出ています。

自分たちの組織がどの業種に属しているかを照らし合わせることで、自社に関連するリスクの高さをざっくり把握できます。

## 2.10 その他のウィジェット

ダッシュボード下部にはさらにいくつかのウィジェットがあります。

**Most Active Malware（直近3ヶ月）**: 円グラフで表示。WizardNet、coruna、DinDoor、FakeSet、StageCompなどが上位。

**Most Active Vulnerabilities（直近3ヶ月）**: CVE番号と関連するレポート数を表示。例えばCVE-2025-55182（92件）、CVE-2025-24893（79件）など。

**Targeted Countries（直近3ヶ月）**: 世界地図上に攻撃の標的国を色分けで表示。北米、ヨーロッパ、東アジアが濃い色で表示されています。
